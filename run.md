# How to Run Vaani

## Prerequisites
- Python 3.11+
- Supabase Account
- Sarvam AI API Key

## Local Setup
1. `cd backend`
2. `python -m venv win_venv`
3. `win_venv\Scripts\activate`
4. `pip install -r requirements.txt`
5. Configure `backend/.env`
6. `python -m uvicorn main:app --reload`

## Frontend
The frontend is served automatically at `http://localhost:8000`.
