# Project Scope — Vaani Smart Data Intelligence

## Included Features
- Programmatic translation of natural language to SQL.
- Manual/Programmatic analysis summary generation.
- Multilingual voice support via Sarvam AI (11 languages).
- Secure data storage in Supabase PostgreSQL.
- Interactive data visualization (Charts/Tables).
- Secure file upload (CSV/Excel).

## Out of Scope
- LLM-based query generation (OpenAI/Anthropic).
- AI-driven speculative data analysis.
- Multi-user authentication (currently single-vault).
